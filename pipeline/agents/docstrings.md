# Docstring Agent

Add or update module-level docstrings in source files.

**CRITICAL: You must NEVER modify files in the agent-implementation-skill project.**

## Method

1. Identify source files in the codespace lacking module-level docstrings
2. For each, write a concise module-level docstring (purpose, key classes/functions)
3. Insert at the top — no other changes

Only add/update docstrings. Skip generated/config files.
